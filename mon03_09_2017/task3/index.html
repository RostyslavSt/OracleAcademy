Задание №3. Функциональное наследование Создайте конструктор объектов Геометрическая фигура, определите в нем приватные свойства для задания центра фигуры. Для доступа к этим свойствам определите необходимые метод/методы. На основе этого конструктора создайте два новых (наследники) – Прямоугольник и Круг. В этих конструкторах определите особые приватные свойства. (Например, длина диагонали для Квадрата и радиус для Круга). В каждый из конструкторов добавьте публичный метод info(), который выводит на экран всю доступную информацию о фигуре.
